# LumenKids

(Este nombre es provisional y no definitivo)

Prototipo inicial del módulo de gamificación académica para un ecosistema educativo inteligente enfocado en aprendizaje autónomo, pensamiento lógico, habilidades tecnológicas y resolución colaborativa de problemas.

Ahora incluye una capa de acceso con login local, perfiles activos y tablero por curso/grupo para visualizar progreso individual y de clase.

El progreso se guarda por perfil en `localStorage` y el panel docente permite editar cursos, grupos, asignaciones y ranking de forma local.

## Qué incluye

- Retos semanales con puntuación y progreso persistente.
- Olimpiada matemática con evaluación por rondas.
- Sistema de roles para cooperación en problemas abiertos.
- Login local con perfiles y selector de curso/grupo.
- Tablero de curso/grupo con indicadores de clase.
- Progreso independiente por perfil.
- Panel docente para editar cursos, grupos, asignaciones y ranking.
- Insignias digitales por constancia, precisión y trabajo en equipo.

## Cómo abrirlo

1. Abre `index.html` en el navegador.
2. Usa los botones para completar retos, resolver la olimpiada y registrar el caso colaborativo.
3. El progreso se guarda en el navegador mediante `localStorage`.
